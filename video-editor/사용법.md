# 수업영상 자동 편집기 (루멘수학)

말이 없는 **조용한 구간(판서 시간·잠깐 멈춤·학생 기다리는 시간 등)을 자동으로 찾아 잘라내는** 프로그램입니다.
자막은 건드리지 않습니다. 캡컷 없이 이것만으로 앞뒤·중간의 불필요한 부분이 정리됩니다.

사용하는 방법은 **두 가지**인데, **① 화면(창) 버전**을 추천합니다.

---

## ① 화면 버전 — `영상편집기` (추천)

창이 뜨고, **자를 부분마다 그 장면 사진이 보여서** 눈으로 확인하며 편집할 수 있습니다.

**쓰는 순서**

1. **`영상편집기`** 를 더블클릭하면 프로그램 창이 열립니다.
2. 왼쪽 위 **📂 영상 열기** 를 눌러 수업영상을 고릅니다.
3. **🔍 무음 분석하기** 를 누릅니다.
   - 잠시 뒤, **자를 구간이 "장면 사진 + 시간 + 체크박스"** 로 쭉 나옵니다.
4. 목록을 봅니다.
   - **체크되어 있음 = 잘라냄** (기본값)
   - **이 부분은 남기고 싶다** → 그 줄의 **체크를 해제**합니다.
   - 위쪽에 "자를 곳 N곳 · 예상 완성 길이"가 실시간으로 바뀝니다.
   - 너무 많이/적게 잡히면 **'자동 감지 설정' 슬라이더**를 움직이고 다시 '무음 분석하기'.
5. **🎬 완성 영상 만들기** 를 누릅니다. 진행 바가 끝나면 완성!
   - 완성본 `영상이름_편집완성.mp4` 가 원본과 같은 폴더에 생깁니다.

> **원본 영상은 절대 손대지 않습니다.** 완성본은 항상 `_편집완성.mp4` 라는 **새 파일**로 만들어집니다.

---

## ② 끌어다 놓기 버전 — `1_분석` / `2_영상만들기`

창 없이 간단히 쓰고 싶을 때. 자를 목록을 **메모장**으로 확인합니다.

1. 영상을 **`1_분석`** 아이콘 위로 끌어다 놓기 → 자를 목록(`..._컷목록.txt`)이 메모장으로 열림
2. 남길 부분은 그 줄 맨 앞에 `#` 을 붙이거나 줄을 지우고 저장
3. 같은 영상을 **`2_영상만들기`** 위로 끌어다 놓기 → 완성본 생성

---

## 처음 한 번만 설치 (윈도우)

프로그램을 돌리려면 두 가지가 필요합니다. **처음 한 번만** 하면 됩니다.

### 가장 쉬운 방법 (윈도우 10·11)

1. 화면 왼쪽 아래 **시작(⊞)** 버튼을 누르고 `cmd` 라고 칩니다.
2. 나오는 **명령 프롬프트**를 **마우스 오른쪽 클릭 → "관리자 권한으로 실행"**.
3. 아래 두 줄을 한 줄씩 붙여넣고 각각 Enter:

   ```
   winget install --id Gyan.FFmpeg -e
   winget install --id Python.Python.3.12 -e
   ```

4. 설치가 끝나면 창을 닫고 **컴퓨터를 한 번 재시작**합니다. (중요)

> 파이썬 설치 시 화면에 **"Add Python to PATH"** 체크박스가 보이면 꼭 켜 주세요.
> (winget으로 설치하면 자동으로 됩니다.)

잘 안 되면 편하게 물어보세요. 설치 화면을 캡처해서 보내주시면 같이 봐 드릴게요.

## 너무 많이/적게 잘릴 때 (감도 조절)

- **화면 버전**: '자동 감지 설정' 슬라이더를 움직인 뒤 다시 '무음 분석하기'.
  값을 **키우면**(예 2.5초) 확실히 긴 정적만 잘라 **덜** 잘립니다.
- **더 세밀하게**: `autocut.py` 파일을 메모장으로 열면 맨 위에 조절용 숫자가 있습니다.

| 항목 | 뜻 | 이렇게 |
|------|----|--------|
| `MIN_SILENCE = 1.5` | 몇 초 이상 조용해야 자를지 | 키우면 덜 잘림 |
| `NOISE_DB = -30` | 어느 정도 작은 소리를 '무음'으로 볼지 | `-25` 민감(많이), `-35` 덜 |
| `KEEP_PADDING = 0.35` | 무음 양쪽으로 남길 여유(초) | 말 끝이 잘리면 키우기(예 0.6) |

## 자주 묻는 것

- **원본이 지워지나요?** 아니요. 완성본은 항상 `_편집완성.mp4` 라는 **새 파일**입니다.
- **한글 파일명 써도 되나요?** 아이폰 → 윈도우로 **바로** 옮기신다면 **안전합니다.**
  (맥을 거치면 한글이 깨질 수 있으니 그때는 영어+숫자 권장.) 예: `2026-07-19_고2_미적분_3강.mp4`
- **mp4 말고 다른 형식도 되나요?** mov, mkv, avi 등 대부분 됩니다. 완성본은 mp4로 나옵니다.
- **여러 영상을 한꺼번에?** 지금은 한 개씩입니다. 필요하면 여러 개 자동 처리도 만들어 드릴게요.
- **검은 창이 잠깐 떠요.** 정상입니다. 화면 버전(`영상편집기`)은 검은 창 없이 프로그램 창만 뜹니다.
