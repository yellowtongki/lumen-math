# 수업영상 자동 편집기 (루멘수학)

말이 없는 **조용한 구간(판서 시간·잠깐 멈춤·학생 기다리는 시간 등)을 자동으로 찾아 잘라내는** 프로그램입니다.
자막은 건드리지 않습니다. 캡컷 없이 이것만으로 앞뒤·중간의 불필요한 부분이 정리됩니다.

편집은 **2단계**로 진행합니다.

1. **1_분석** — 영상을 넣으면 자를 곳 목록(`..._컷목록.txt`)을 만들어 보여줍니다.
2. (원장님이 목록을 눈으로 확인/수정)
3. **2_영상만들기** — 확인한 목록대로 잘라낸 **완성 영상**(`..._편집완성.mp4`)을 만듭니다.

> 원본 영상은 **절대 손대지 않습니다.** 새 파일(`_편집완성.mp4`)로 따로 만들어지니 안심하세요.

---

## 처음 한 번만 설치 (윈도우)

프로그램을 돌리려면 두 가지가 필요합니다. **처음 한 번만** 하면 됩니다.

### 방법 A — 자동 설치 (윈도우 10·11, 가장 쉬움)

1. 화면 왼쪽 아래 **시작(⊞)** 버튼을 누르고 `cmd` 라고 칩니다.
2. 나오는 **명령 프롬프트**를 **마우스 오른쪽 클릭 → "관리자 권한으로 실행"**.
3. 아래 두 줄을 한 줄씩 붙여넣고 각각 Enter:

   ```
   winget install --id Gyan.FFmpeg -e
   winget install --id Python.Python.3.12 -e
   ```

4. 설치가 끝나면 창을 닫고 **컴퓨터를 한 번 재시작**합니다. (중요)

### 방법 B — 직접 설치

- **ffmpeg**: https://ffmpeg.org/download.html 에서 윈도우용을 받아 설치 후, 시스템 PATH에 등록.
- **Python**: https://www.python.org/downloads/ 에서 설치. 설치 첫 화면의
  **"Add Python to PATH"** 체크박스를 꼭 켜세요.

> 잘 안 되면 편하게 물어보세요. 캡처 화면 보내주시면 같이 봐 드릴게요.

## 사용법 (매번)

이 `video-editor` 폴더를 통째로 컴퓨터에 두고 씁니다.

1. 편집할 영상 파일(예: `수업.mp4`)을 준비합니다.
2. 그 영상을 **`1_분석`** 아이콘 위로 **마우스로 끌어다 놓습니다** (드래그 앤 드롭).
   - 잠시 뒤 자를 곳 목록(`수업_컷목록.txt`)이 **메모장으로 저절로 열립니다.**
3. 목록을 확인합니다.
   - 각 줄 = **"이 구간을 잘라낸다"**.
   - **이 부분은 남기고 싶다** → 그 줄 맨 앞에 `#` 을 붙이거나, 그 줄을 지웁니다.
   - 자를 시간을 조금 바꾸고 싶다 → 숫자만 직접 고칩니다 (형식: `시:분:초.소수`).
   - 다 봤으면 **저장**(Ctrl+S)하고 메모장을 닫습니다.
4. 같은 영상을 **`2_영상만들기`** 아이콘 위로 끌어다 놓습니다.
   - 완성본 `수업_편집완성.mp4` 가 같은 폴더에 만들어집니다. (영상 길이에 따라 몇 분 걸릴 수 있어요)
5. 완성본을 VOD에 올리면 끝!

## 너무 많이/적게 잘릴 때 (감도 조절)

`autocut.py` 파일을 메모장으로 열면 맨 위에 조절용 숫자가 있습니다. 여기 숫자만 바꾸면 됩니다.

| 항목 | 뜻 | 이렇게 바꾸면 |
|------|----|--------------|
| `MIN_SILENCE = 1.5` | 몇 초 이상 조용해야 자를지 | 값을 **키우면**(예 2.5) 확실히 긴 정적만 잘라 **덜** 잘림 |
| `NOISE_DB = -30` | 어느 정도 작은 소리를 '무음'으로 볼지 | `-25`로 하면 더 민감(=많이 잘림), `-35`면 덜 잘림 |
| `KEEP_PADDING = 0.35` | 무음 양쪽으로 남길 여유(초) | 말 끝이 잘리면 **키우세요**(예 0.6) |

> 조절 후에는 다시 `1_분석`부터 하면 됩니다.

## 자주 묻는 것

- **원본이 지워지나요?** 아니요. 완성본은 항상 `_편집완성.mp4` 라는 **새 파일**입니다.
- **여러 영상을 한꺼번에?** 지금은 한 개씩 하시면 됩니다. 필요하면 여러 개 자동 처리도 만들어 드릴게요.
- **mp4 말고 다른 형식도 되나요?** mov, mkv, avi 등 대부분 됩니다. 완성본은 mp4로 나옵니다.
- **중간에 딱 한 군데만 자르고 싶어요.** `1_분석` 없이 컷목록에 직접
  `00:03:10 - 00:05:00` 처럼 한 줄 적고 `2_영상만들기` 해도 됩니다.
